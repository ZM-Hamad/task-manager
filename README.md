# Tasks App

## Tech Stack
- React + TypeScript
- React Router
- Node.js + Express
- MongoDB
- JWT Authentication

## Features
- Register / Login
- Protected routes
- Tasks CRUD (Create, Read, Update, Delete)
- Pagination
- Auth-based navigation
- Auto logout on unauthorized

## How to Run
### Backend
cd server
npm install
npm run dev

### Frontend
cd client
npm install
npm run dev
