# Tasks App

## Features
- Register / Login
- Protected routes
- Tasks CRUD (Create, Read, Update, Delete)
- Pagination
- Auto logout on unauthorized
- Auth-based navigation

## Tech Stack
- React + TypeScript
- React Router
- Node.js + Express
- MongoDB
- JWT Authentication

## How to Run

### Backend
cd server
npm install
npm run dev

### Frontend
cd client
npm install
npm run dev
